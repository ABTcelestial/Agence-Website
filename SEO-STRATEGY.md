## E-E-A-T (Expertise, Expérience, Autorité, Fiabilité)

### Actions Prioritaires

- [ ] Créer page équipe `/about/equipe` avec photos professionnelles, bios détaillées, liens LinkedIn
- [ ] Ajouter schema `Person` avec `sameAs` LinkedIn pour chaque membre
- [ ] Publier 3 études de cas avec métriques clients algériens (avant/après)
- [ ] Compléter le schema `LocalBusiness` : adresse, téléphone, horaires
- [ ] Inscription dans les annuaires algériens (YallaLouise, Annuaire Algérie Pro)
- [ ] Créer Google Business Profile (google.com/business)
- [ ] Ajouter schéma `FAQPage` sur chaque page de service (4-6 questions)

### Signaux de confiance à afficher

- Nombre de clients servis
- Années d'expérience
- Logos clients (avec autorisation)
- Témoignages avec nom, poste, entreprise
- Certifications ou formations (si applicable)

## GEO — Optimisation pour Moteurs Génératifs

Le robots.txt actuel autorise déjà GPTBot, PerplexityBot, ClaudeBot, anthropic-ai — à maintenir.

- [ ] Créer `llms.txt` à la racine avec description structurée des services pour IA
- [ ] Structurer les études de cas avec métriques citables ("temps de traitement réduit de 60%")
- [ ] Reformater les titres de blog en questions directes avec réponses extractibles
- [ ] Ajouter `FAQPage` schema sur les pages services (pas seulement le blog)
- [ ] Maintenir la cohérence des entités (nom, adresse, téléphone) sur toutes les pages

---

## Lacunes Techniques à Corriger (Priorité Haute)

1. **Hreflang manquant** — aucun tag alternatif pour fr/ar/en sur les pages
2. **LocalBusiness incomplet** — adresse et téléphone absents du schema
3. **Pas de `llms.txt`** — manque pour les crawlers IA
4. **FAQPage schema** — présent uniquement dans les articles de blog, pas sur les services
5. **Catégories blog** — pas de pages index par catégorie (`/blog/categorie/*`)
